Git و GitHub

كن أنت من يتحكم في تاريخ مشروعك

البرمجة ليست مجرد كتابة أكواد
بل هي أيضًا مراجعة وتطوير وتراجع وتصحيح وتعاون
وفي هذا المسار المتكرر، لا يوجد أعظم من Git كأداة تنظّم كل هذه الخطوات وتمنحك القدرة على إدارة تاريخ مشروعك كاملًا
ثم جاء GitHub ليجعل من ذلك التاريخ مشروعًا حيًا قابلًا للمشاركة والتطوير الجماعي

ما هو Git؟

Git هو نظام "تحكم في الإصدارات" (Version Control System)
ابتكره لينوس تورفالدز (مخترع لينكس) عام 2005
فكرته الأساسية بسيطة وعظيمة:
كل تغيّر في كودك يتم حفظه كتاريخ، ويمكنك الرجوع لأي نقطة في هذا التاريخ، كما يمكنك مقارنة ما تغيّر، ومن غيّره، ولماذا

ماذا تفعل Git كمبرمج؟

تحفظ كل تعديل تقوم به على ملفاتك مع رسالة توضيحية

تُتيح لك العودة إلى أي لحظة سابقة في المشروع كأنك تسافر عبر الزمن

تُجنّبك الكوارث الناتجة عن الحذف أو التعديل غير المقصود

تُشجع على التفكير المنظّم والوعي الذاتي بكل سطر تكتبه

ما هو GitHub؟

GitHub هو منصة على الإنترنت تستضيف مشاريع Git
وتُتيح للمبرمجين من مختلف أنحاء العالم أن يعملوا معًا
يراجع بعضهم أعمال بعض، يكتبون الملاحظات، يقترحون التحسينات
كما أنها تُستخدم اليوم كسيرة ذاتية حيّة للمبرمج
لأن أي شركة تستطيع رؤية طريقة عملك، لا ما تدّعي أنك تعرفه

مزايا Git و GitHub:

حفظ تلقائي لتاريخ المشروع

القدرة على الرجوع لأي نسخة سابقة بدون قلق

التفرع (Branches): جرب أفكارًا جديدة دون أن تخرب الأصل

الدمج (Merge): اجمع الأفكار بعد اختبارها

التعاون: أكثر من شخص يعمل على نفس المشروع بسلام وتنظيم

الشفافية: كل شيء موثق بالأسماء والتواريخ

التنقل الآمن بين البيئات: من التطوير إلى الإنتاج بسلاسة

عيوبه أو حدوده:

يحتاج لتعلّم أولي بسيط (بعض الأوامر الأساسية)

سطر الأوامر قد يبدو مرعبًا للمبتدئ في البداية

الدمج (Merge) قد يسبب تعارضات يصعب حلها إذا لم تكن منظمًا

GitHub ليس أداة Git الوحيدة، بل مجرد مستضيف (يوجد GitLab و Bitbucket وغيرهما)

كيف تستخدم Git باحتراف؟

ابدأ دائمًا بأمر git init داخل مجلد مشروعك

راجع تعديلاتك باستخدام git status و git diff

احفظ نسختك بتغييراتها عبر git commit -m "شرح التعديل"

أنشئ فرعًا جديدًا عند تجربة ميزة عبر git branch feature-x ثم git checkout feature-x

استخدم GitHub لتخزين نسختك في السحابة وتشاركها مع غيرك

تعلم استخدام Pull Requests و Code Reviews لتتطور مهنيًا وفكريًا

لمن تناسب هذه الأداة؟

للمبرمج الفردي حتى لو لم يعمل مع فريق

للفرق التي تطور مشاريعًا مشتركة

لكل من يريد أرشفة شغله وفهم تطوره مع الوقت

لكل من يفكر باحتراف

الحكمة: من حفظ تاريخه لم يخشَ مستقبله

بقلم: شهاب أوزباشي أوغلو
